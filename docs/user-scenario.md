# Користувацький сценарій

## Умови початку
Користувач відкриває веб-застосунок FitBook.

## Кроки дій
1. Ввести назву вправи, кількість підходів і повторень.  
2. Натиснути кнопку "Додати замітку".  
3. Замітка з'являється у списку.  
4. Користувач може відмітити замітку виконаною.  
5. Користувач може редагувати або видаляти замітку.

## Очікуваний результат
- Замітка додана у список.  
- Відмітка про виконання працює.  
- Функції редагування та видалення доступні та працюють без помилок.
