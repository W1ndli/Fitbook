# High-Level Overview

## Епіки
- EPIC-01: Управління замітками (US-01, US-02, US-03, US-05)
- EPIC-02: Перегляд заміток (US-04)
- EPIC-03: Збереження даних (US-06)

## Матриця трасування FR → US
| FR | Відповідні US |
|----|---------------|
| FR-01 | US-01 |
| FR-02 | US-02 |

| FR-05 | US-05 |

## Найважливіші NFR
- NFR-01 (Продуктивність): ≤2 с для 95% запитів, перевірка через тест навантаження, ок = виконано
- NFR-02 (Надійність): замітки зберігаються між сесіями, тест через закриття/відкриття застосунку, ок = виконано
- NFR-03 (Безпека): доступ лише авторизованих користувачів, тест через спробу доступу без логіну, ок = відмовлено у доступі

## Ризики
- R-01: Втрата даних → зменшити через локальне та хмарне збереження.
- R-02: Некоректне відображення списку → тестування на різних браузерах.
- R-03: Повільна робота при великій кількості заміток → обмеження до 100 заміток/користувач + оптимізація запитів.
