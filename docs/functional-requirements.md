# Функціональні вимоги (FR)

| ID    | Вимога |
|-------|--------|
| FR-01 | Система повинна дозволяти користувачу створювати замітку з вправою. |
| FR-02 | Система повинна дозволяти редагувати замітку. |
| FR-03 | Система повинна дозволяти видаляти замітку. |
| FR-04 | Система повинна відображати список усіх заміток. |
| FR-05 | Система повинна дозволяти відмічати замітку як виконану. |
| FR-06 | Система повинна зберігати замітки між сесіями користувача. |
