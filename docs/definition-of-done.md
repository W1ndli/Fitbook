# Definition of Done (DoD)

1. User story реалізована згідно FR.
2. Всі Acceptance Criteria виконані.
3. Код пройшов базові тести без критичних помилок.
4. Замітки зберігаються та відображаються правильно.
5. Статус замітки зберігається між сесіями.
6. Документація оновлена (FR, NFR, AC).
7. Інтерфейс функціонує у сучасних браузерах (Chrome, Firefox, Edge).
