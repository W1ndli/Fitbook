# Нефункціональні вимоги (NFR)

| ID      | Атрибут       | Вимога / Метрика |
|---------|---------------|-----------------|
| NFR-01  | Продуктивність | Час відповіді при додаванні замітки ≤2 с для 95% запитів |
| NFR-02  | Надійність    | Замітки зберігаються при закритті та повторному відкритті застосунку |
| NFR-03  | Безпека       | Доступ лише авторизованих користувачів; тест: спроба доступу без логіну |
| NFR-04  | Масштабованість | Система підтримує до 100 заміток на одного користувача |
| NFR-05  | Зручність     | Інтерфейс інтуїтивно зрозумілий, перевіряється тестуванням користувачів |
| NFR-06  | Сумісність    | Працює на сучасних браузерах: Chrome, Firefox, Edge |
