# Acceptance Criteria

## US-01
- AC-01: Given користувач ввів дані → When натискає "Додати" → Then замітка з’являється у списку
- AC-02: Замітка зберігається при оновленні сторінки

## US-02
- AC-01: Given існуюча замітка → When користувач редагує її → Then зміни відображаються у списку
- AC-02: Немає втрати інших заміток

## US-03
- AC-01: Given існуюча замітка → When користувач видаляє → Then замітка зникає зі списку
- AC-02: Система не видаляє інші замітки

## US-04
- AC-01: Список відображається коректно з усіма замітками
- AC-02: Можна сортувати за датою або назвою

## US-05
- AC-01: Користувач відмічає замітку → Then відмітка зберігається
- AC-02: Відмітка видима у списку

## US-06
- AC-01: Замітки зберігаються при перезавантаженні сторінки
- AC-02: Замітки доступні при повторному вході користувача
