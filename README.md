# FitBook

## Пітч продукту
FitBook — веб-застосунок для створення власних заміток з тренуваннями. Користувач може самостійно прописувати вправи, кількість підходів і повторень, відмічати виконане і переглядати список заміток. Це дозволяє систематизувати тренування і відстежувати прогрес.

## MVP
Мінімально життєздатний продукт дозволяє:
- Додати замітку з вправою.
- Переглянути всі замітки у списку.
- Відмітити замітку як виконану.

## Стейкхолдери
- Користувачі: самостійні спортсмени та початківці.  
- Викладач або ментор: оцінює якість MVP і прогрес.  
- Розробник (ти): реалізує функціонал та вдосконалює продукт.

## Мета першої ітерації
Створити базовий функціонал заміток для тренувань, який можна перевірити на користувачах.

## Ознаки успіху
- Користувач може додати замітку.  
- Замітка відображається у списку.  
- Замітка може бути відмічена як виконана.
